# Security Policy

Voice2Sub is a local-first desktop app for subtitle generation and transcription workflows.

## Reporting a vulnerability

Please do not open a public GitHub issue for sensitive security reports.

Contact the project owner through the official website:

https://voice2sub.pro.vn/

Include:

- A clear description of the issue
- Steps to reproduce
- Affected version and platform
- Any relevant logs or proof of concept

## Privacy note

Voice2Sub is designed around local subtitle/transcription workflows. Users should still avoid sharing private media files in public issue reports.
