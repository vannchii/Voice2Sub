# Support

For Voice2Sub support, please use one of these options:

- Website: https://voice2sub.pro.vn/
- Download page: https://voice2sub.pro.vn/download
- What's new: https://voice2sub.pro.vn/whats-new
- GitHub Issues: use the bug report or feature request templates in this repository.

When reporting a problem, please include:

- Voice2Sub version
- Operating system and version
- CPU/GPU information if the issue relates to CUDA, Metal, performance or model execution
- What file type you used, if relevant
- Steps to reproduce
- Screenshot or log excerpt if available

Please do not upload private audio/video files publicly unless you are sure they are safe to share.
